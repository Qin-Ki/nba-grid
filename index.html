How to Play

🎯 Goal

Fill the 3×3 grid with NBA players who match both the row and column criteria for each cell.

🎮 How to Play

  1. Click any empty cell to search for a player
  2. Type a player's name and select from the results
  3. Each guess counts — you have 9 total (one per cell)
  4. A player can only be used once per grid

📊 Scoring

  • Correct answers: X out of 9
  • Rarity score: Lower percentage = more obscure = better
  • Rarity = how often other players chose that answer (%)

🏀 规则

  • 网格的 行和列都是 NBA 球队
  • 每个格子需要填入一位 效力过这两支球队 的球员
  • 球队历史合并计算（例如雷霆包含超音速）
